TransferTransaction 

Companion object TransferTransaction

case class TransferTransaction[P <: Proposition](from: IndexedSeq[(Address, Long)], to: IndexedSeq[(Address, TokenValueHolder)], attestation: Map[P, Proof[P]], fee: Long, timestamp: Long, data: Option[String], minting: Boolean, txType: String)(implicit evidence$1: EvidenceProducer[P], evidence$2: Identifiable[P]) extends Product with Serializable

A transfer transaction which can be a: poly transaction, arbit transaction or asset transaction

P

- Proposition type (PublicKeyProposition or ThresholdProposition)

from

- list of senders' addresses and box nonce's used for the transaction

to

- list of address and token values to send to

attestation

- mapping of proposition to signature

fee

- transaction fee

timestamp

- the current time (as the number of milliseconds from the epoch)

data

- data for the transaction

minting

- if the transaction is a minting transaction, minting = true (for asset transfers only)

txType

- type of transaction: "PolyTransfer", "AssetTransfer", or "ArbitTransfer"
