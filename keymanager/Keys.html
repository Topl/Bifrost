Keys 

Companion object Keys

class Keys[S <: Secret, KF <: Keyfile[S]] extends Logging

The keys class is equivalent to the KeyRing in Bifrost. It holds the current keys in the wallet application and handles functions on the keys such as: locking/unlocking, generating a new key, etc.

S

the type of Secret for these keys

KF

the type of Keyfile (crypto.KeyfileCurve25519)
