PeerSpec

case class PeerSpec(agentName: String, version: Version, nodeName: String, declaredAddress: Option[InetSocketAddress], features: Seq[PeerFeature]) extends Product with Serializable

Declared information about peer

agentName

Network agent name. May contain information about client code stack, starting from core code-base up to the end graphical interface. Basic format is /Name:Version(comments)/Name:Version/.../, e.g. /Ergo-Scala-client:2.0.0(iPad; U; CPU OS 3_2_1)/AndroidBuild:0.8/

version

Identifies protocol version being used by the node

nodeName

Custom node name

declaredAddress

Public network address of the node if any

features

Set of node capabilities
